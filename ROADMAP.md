# Roadmap

Cloak Browser is focused on authorized QA workflows for browser, network,
locale, timezone, WebRTC, and Playwright consistency.

## v1 Priorities

- Improve scan accuracy and explain the highest-impact mismatch first.
- Make Playwright preset export and CI threshold checks easy to adopt.
- Expand practical preset coverage across common desktop, mobile, browser, and
  region combinations.
- Keep shared reports privacy-safe, redacted, short-lived, and useful for team
  handoff.

## Deferred

- npm package publishing beyond the existing package metadata.
- Self-hosting templates and deployment variants.
- Proxy-provider-specific presets or workflows.
- Any guidance for bypassing authentication, CAPTCHA, bot controls, fraud
  systems, or systems the operator is not authorized to test.
